## 🎨 Design System & Visual Strategy

### 1. Color Palette & Rationale
*   **Deep Charcoal / Slate Blue Gradient:** 
    *   *Why I used it:* Serves as a muted, professional background that allows foreground text and portrait photography to pop. It gives a modern, sleek "dark mode" tech aesthetic suitable for a UI/UX showcase.
*   **Warm Mustard Gold / Amber Text accents:** 
    *   *Why I used it:* Acts as an intentional focal point identifier. It breaks the dark monotone layout, drawing immediate attention to key headers, quotes ("Giving The Best"), and personal branding names.
*   **Crisp White:** 
    *   *Why I used it:* Ensures high readability and strict visual hierarchy against the dark background canvas.

### 2. Graphic Elements & Layout Choices
*   **Dynamic Sinusoidal Wave Lines (Table of Contents):** 
    *   *Why I used it:* The neon purple-to-cyan gradient wave acts as a directional flow element guiding the eye from the portrait photo towards the navigation menu blocks.
*   **Geometric Framing Triangles (My Design Slide):** 
    *   *Why I used it:* Interlocking white and gold wireframe triangles are placed in the background to dynamically section off three distinct project phases (Figma prototyping, app redesign, interface redesign), adding structural complexity without cluttering the screen.
*   **Rounded Capsule Blocks (Expertise & Resume Cards):** 
    *   *Why I used it:* Employs frosted glass-like light grey cards with rounded corners to separate information neatly, adopting standard mobile and web component UI conventions.
